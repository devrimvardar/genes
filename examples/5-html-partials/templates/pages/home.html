Welcome to HTML Partials Demo

This page demonstrates how Genes Framework can work entirely with static HTML partial files - no database, no complex templating engine, just simple file includes.

How It Works

  • Each page is composed from multiple HTML partial files
  • Variables are replaced at runtime using simple {{variable}} syntax
  • No database connection needed
  • Perfect for marketing sites, documentation, and portfolios
  • Easy for designers to edit - pure HTML!
