Get Started with genes

Start building in 60 seconds. No installation, no package managers, no build tools.

Method 1: Clone the Repository

git clone https://github.com/devrimvardar/genes.git

View on GitHub

Method 2: Download Files Directly

Download just the core files you need:

  • genes.php (~5,900 lines)
  • genes.css (~1,700 lines)
  • genes.js (~1,300 lines)

Quick Setup

  1. Create a project folder
  2. Place genes.php, genes.css, genes.js in the folder
  3. Create index.php with <?php require_once "genes.php";
  4. Create data/config.json for configuration
  5. Open in your browser - you're done!

Read the full setup guide →

Requirements

  • PHP 7.4+ (no extensions required)
  • Any web server (Apache, Nginx, PHP built-in server)
  • Modern web browser for the UI framework
