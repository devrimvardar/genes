Core Concept

Genes uses a simple function-based API through the g::run() method:

// Connect to database
g::run("db.connect", array("driver" => "mysql", "database" => "mydb"));

// Create schema (5 tables: clones, persons, items, labels, events)
g::run("db.createSchema");

// Set current clone for multi-tenant isolation
g::run("db.setClone", "abc123");

// CRUD operations
g::run("db.createItem", array("name" => "Example Item"));
$items = g::run("db.getItems");
g::run("db.updateItem", array("id" => 1, "name" => "Updated"));
g::run("db.deleteItem", 1);

Database Schema (5 Tables)

clones        - Master tenant table (id, hash, data)
  ‚îî‚îÄ persons  - Users/contacts (id, clone_id, data)
  ‚îî‚îÄ items    - Generic entities (id, clone_id, data)
  ‚îî‚îÄ labels   - Categories/tags (id, clone_id, data)
  ‚îî‚îÄ events   - Activity log (id, clone_id, data)

No Database? No Problem!

Use config.json for data storage or HTML partials for static sites:

// Load configuration
$config = json_decode(file_get_contents("config.json"), true);

// Load HTML partial with variable substitution
function loadPartial($file, $vars = array()) {
    $content = file_get_contents($file);
    foreach ($vars as $key => $value) {
        $content = str_replace("{{" . $key . "}}", $value, $content);
    }
    return $content;
}

Multi-Tenant Isolation

// Set current clone (tenant)
g::run("db.setClone", "customer-abc");

// All queries are now automatically filtered to this clone
$items = g::run("db.getItems"); // Only returns items for customer-abc

// Switch to different clone
g::run("db.setClone", "customer-xyz");
$items = g::run("db.getItems"); // Returns items for customer-xyz

üîß Common Tasks

Connect to MySQL

g::run("db.connect", array(
    "driver" => "mysql",
    "host" => "localhost",
    "database" => "mydb",
    "username" => "user",
    "password" => "pass"
));

Connect to SQLite

g::run("db.connect", array(
    "driver" => "sqlite",
    "database" => "data/app.db"
));

Create a Clone

$clone = g::run("db.createClone", array(
    "hash" => "unique-id",
    "name" => "Client Name"
));

Create an Item

g::run("db.setClone", "unique-id");
$item = g::run("db.createItem", array(
    "name" => "My Item",
    "description" => "Details"
));

Query Items

$items = g::run("db.getItems", array(
    "where" => "name LIKE '%search%'",
    "orderby" => "created DESC",
    "limit" => 10
));

Build REST API

$method = $_SERVER['REQUEST_METHOD'];
if ($method === 'GET') {
    echo json_encode(g::run("db.getItems"));
} elseif ($method === 'POST') {
    $data = json_decode(file_get_contents('php://input'), true);
    echo json_encode(g::run("db.createItem", $data));
}

üíª System Requirements

  • PHP: 5.6 or higher (tested up to PHP 8.x)
  • Database: MySQL 5.5+ or SQLite 3.x (or none!)
  • Server: Apache, Nginx, or any PHP-capable web server
  • Dependencies: None! No Composer, npm, or build tools required
