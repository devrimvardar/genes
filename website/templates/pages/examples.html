2. Multi-Tenant Blog

View Source ‚Üí

Blog platform with clone-based isolation. Each blog (clone) has its own posts, comments, and categories - all automatically filtered by clone_id.

Multi-Tenant Clones Blog Isolation

Key Concepts:

// Create a blog (clone)
$blog = g::run("db.createClone", array("hash" => "tech-blog", "name" => "Tech Blog"));

// Set current blog
g::run("db.setClone", "tech-blog");

// All operations are now isolated to this blog
g::run("db.createItem", array("title" => "My Post", "content" => "..."));
$posts = g::run("db.getItems"); // Only returns tech-blog posts

// Switch to different blog
g::run("db.setClone", "personal-blog");
$posts = g::run("db.getItems"); // Returns personal-blog posts

3. RESTful API

View Source ‚Üí

Full REST API with GET, POST, PUT, DELETE endpoints. Includes JSON responses, error handling, and multi-tenant support via API keys.

REST API JSON HTTP Methods API Keys

Key Concepts:

// Route based on HTTP method
$method = $_SERVER['REQUEST_METHOD'];

if ($method === 'GET') {
    // List all items
    $items = g::run("db.getItems");
    echo json_encode(array("success" => true, "data" => $items));
    
} elseif ($method === 'POST') {
    // Create new item
    $data = json_decode(file_get_contents('php://input'), true);
    $item = g::run("db.createItem", $data);
    echo json_encode(array("success" => true, "data" => $item));
}

5. HTML Partials

View Source ‚Üí

Build static sites using HTML partial templates with variable substitution. No database, no build tools - just clean HTML composition.

Templates Partials Static Sites Variables

Key Concepts:

// Load partial with variable substitution
function loadPartial($filename, $vars = array()) {
    $content = file_get_contents("templates/" . $filename);
    foreach ($vars as $key => $value) {
        $content = str_replace("{{" . $key . "}}", $value, $content);
    }
    return $content;
}

// Use partials
echo loadPartial("header.html", array("title" => "Welcome"));
echo loadPartial("content.html", array("name" => "John"));
echo loadPartial("footer.html", array("year" => date("Y")));
