Installation Guide

New Project

Add to Existing Project

1

Clone the Repository

git clone https://github.com/devrimvardar/genes.git
cd genes
2

Choose Your Starting Point

Copy one of the examples as your starting point:

# For database project
cp -r examples/1-database-crud my-project

# For static site
cp -r examples/5-html-partials my-project

# Or start from scratch with just the core files
mkdir my-project
cp genes.php genes.js genes.css my-project/
3

Start Development Server

cd my-project
php -S localhost:8000

Open http://localhost:8000 in your browser

4

Start Coding!

Create index.php:

<?php
require_once 'genes.php';

// Connect to database (optional)
g::run("db.connect", array(
    "driver" => "sqlite",
    "database" => "data/app.db"
));

// Create schema
g::run("db.createSchema");

// Your code here
echo "Hello Genes!";
?>
1

Copy Core Files

# Download and copy to your project
curl -O https://raw.githubusercontent.com/devrimvardar/genes/main/genes.php
curl -O https://raw.githubusercontent.com/devrimvardar/genes/main/genes.js
curl -O https://raw.githubusercontent.com/devrimvardar/genes/main/genes.css
2

Include in Your Project

<?php
// Add to your existing PHP files
require_once 'genes.php';

// Use Genes features alongside your existing code
g::run("db.connect", $your_db_config);
?>
3

Configure Database (Optional)

If using database features:

// Use existing MySQL connection
g::run("db.connect", array(
    "driver" => "mysql",
    "host" => "localhost",
    "database" => "your_database",
    "username" => "your_user",
    "password" => "your_password"
));

// Or use SQLite
g::run("db.connect", array(
    "driver" => "sqlite",
    "database" => "path/to/database.db"
));

Quick Start Examples

üóÑÔ∏è With Database (SQLite)

<?php
require_once 'genes.php';

// Connect
g::run("db.connect", array(
    "driver" => "sqlite",
    "database" => "data/app.db"
));

// Create schema
g::run("db.createSchema");

// Create item
g::run("db.createItem", array(
    "name" => "My First Item",
    "description" => "Hello Genes!"
));

// Get all items
$items = g::run("db.getItems");
print_r($items);
?>

üìÑ Without Database (Config)

<?php
require_once 'genes.php';

// Load config data
$config = json_decode(file_get_contents("config.json"), true);

// Use the data
echo "<h1>" . $config['site']['name'] . "</h1>";

foreach ($config['pages'] as $page) {
    echo "<a href='" . $page['url'] . "'>";
    echo $page['title'] . "</a>";
}
?>

üé® With HTML Partials

<?php
require_once 'genes.php';

// Helper function
function loadPartial($file, $vars = array()) {
    $html = file_get_contents("templates/" . $file);
    foreach ($vars as $key => $value) {
        $html = str_replace("{{" . $key . "}}", $value, $html);
    }
    return $html;
}

// Build page
echo loadPartial("header.html", array("title" => "Welcome"));
echo loadPartial("content.html", array("name" => "World"));
echo loadPartial("footer.html", array("year" => date("Y")));
?>

üè¢ Multi-Tenant App

<?php
require_once 'genes.php';

g::run("db.connect", array("driver" => "sqlite", "database" => "app.db"));
g::run("db.createSchema");

// Create tenant (clone)
g::run("db.createClone", array(
    "hash" => "client-abc",
    "name" => "Client ABC Corp"
));

// Set current tenant
g::run("db.setClone", "client-abc");

// All operations isolated to this tenant
g::run("db.createItem", array("name" => "ABC Item"));
$items = g::run("db.getItems"); // Only client-abc items
?>
